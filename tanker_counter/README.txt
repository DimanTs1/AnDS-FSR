# Satellite Ship Detection (Поиск танкеров на спутниковых снимках)

Как это работает
1. Полигональная маска: Программа использует жестко заданные координаты для точного очерчивания пролива. Весь городской шум на материках игнорируется за счет этого.
2. Бинаризация: Оставшиеся в зоне воды пиксели фильтруются по порогу яркости, оставляя только самые яркие точки.
3. Заливка: Алгоритм проходит по найденным точкам и объединяет соседние пиксели в единые объекты (кластеры). 
4. Фильтрация по размеру: Если пятно слишком маленькое (цифровой шум) или слишком большое (кусок берега, надпись на карте), оно отсекается. Подходящие объекты помечаются красным маркером.

Технологии
* Язык: C
* Библиотека: `lodepng` (для чтения и записи PNG изображений без внешних зависимостей)

Как запустить
1. Поместите исходное ЧЕРНО-БЕЛОЕ изображение в корень папки под названием `strait_image.png` (Изображение уже лежит в папке вместе с результатом поиска 'result.png'. Этот файл можно удалить, а изображение заменить на аналогичное, но, например, с повышенной точностью)
2. Запустите код
